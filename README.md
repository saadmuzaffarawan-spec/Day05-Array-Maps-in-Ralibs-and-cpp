# 🟥 Raylib Ghost Bounce Demo

A minimal Raylib + C++ program that demonstrates:

Struct-based game entity

Frame-independent movement using delta time

Proper axis-based wall collision

Direction reflection (bounce physics)

Dynamic color change on each wall hit

The ghost moves diagonally across the screen and bounces realistically off all four boundaries.

# 🎮 Features

Smooth motion using GetFrameTime()

Correct X/Y collision handling

Speed remains constant after bouncing

Clean struct design for game objects

Color feedback for each wall collision

# 🧠 What This Project Teaches

This is a beginner gameplay logic exercise focused on:

Position + direction vectors

Delta-time movement

AABB screen bounds checking

Velocity inversion per axis

Basic game loop structure in Raylib

# 🖥️ Controls

No input required.
The ghost moves and bounces automatically.
